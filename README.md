# polaris
北极星曾经对地球说过，如果你什么时候迷路了，就抬头看看我，我会永远在这儿等你。polaris也是如此，指引着你永远朝着正确的方向不断前行。
北极星计划旨在打造一个知识共享平台，愿集体智慧的星光可与太阳争辉。
